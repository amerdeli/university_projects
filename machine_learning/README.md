# Machine Learning Assignments
This directory contains two university assignments completed for the course *Machine Learning for AIE (TU Graz, WS2025)*.  
Each assignment covers a different set of machine learning methods and includes the corresponding code used to reproduce the results.

## Assignment Overview

### 1. Linear & Logistic Regression, Gradient Descent
Implementation of regression and classification fundamentals on tabular datasets.  
Tasks include univariate and multiple linear regression, polynomial regression, logistic regression with design matrix construction, and gradient descent optimization on the Rastrigin function.

### 2. Course Project — Multi-Class Classification & Outlier Detection
A full machine learning project on a real-world sensor dataset with 9000 samples and 12 features.  
Tasks include exploratory data analysis, stratified K-Fold cross-validation, baseline KNN classifier, experimentation across 6 model families (Logistic Regression, SVM, Random Forest, Gradient Boosting, XGBoost, MLP), and unsupervised outlier detection using GMM, LOF, and Isolation Forest.
