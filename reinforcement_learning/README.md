# Reinforcement Learning Assignments
This directory contains university assignments completed for the course **Reinforcement Learning for AIE (TU Graz)**.  
Each assignment focuses on a different topic in reinforcement learning and includes the corresponding code used to reproduce the results.

## Assignment Overview

### 1. Monte Carlo Simulation, Markov Decision Process & Iterative Policy Evaluation
An introduction to core RL concepts through analytical and numerical methods.  
Tasks include Monte Carlo estimation, solving a two-state MDP using the Bellman equation, and implementing Policy Iteration on the FrozenLake environment from OpenAI Gymnasium.

## Structure

Each assignment folder contains a single Python or Jupyter Notebook file and its own README.

All code files reproduce the results shown in the corresponding reports submitted for grading.
