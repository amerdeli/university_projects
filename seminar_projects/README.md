# Seminar Projects
This directory contains research and implementation projects completed as part of the **AI extension programme at TU Graz**.  
Each project focuses on a specific research topic and includes the corresponding code and notebooks.

## Project Overview

### 1. Neural Networks with Noisy Weights
An investigation into the effect of additive weight noise during training of fully connected neural networks.  
Topics include noisy weight formulation, impact on training and test performance across different architectures, input sensitivity analysis via gradient computation, and comparison of ReLU vs tanh nonlinearities on the MNIST dataset.
